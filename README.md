# Приложение для заметок

Полнофункциональное приложение для заметок на Python с GUI на PyQt5 и локальным хранением в SQLite.

## Возможности

- ✅ Создание, редактирование и удаление заметок
- ✅ Поиск заметок по заголовку и содержимому
- ✅ Сортировка заметок по дате изменения
- ✅ Синхронизация с локальным файлом (JSON)
- ✅ Обработка конфликтов при синхронизации
- ✅ Минималистичный и удобный интерфейс
- ✅ Логирование всех действий

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

## Использование

### Основные функции

- **Добавить**: Создать новую заметку
- **Редактировать**: Изменить выбранную заметку
- **Удалить**: Удалить выбранную заметку
- **Синхронизация**: Синхронизировать заметки с локальным файлом `notes_sync.json`

### Поиск

Введите текст в поле поиска для фильтрации заметок по заголовку или содержимому.

### Синхронизация

При нажатии кнопки "Синхронизация":
1. Заметки загружаются из/сохраняются в файл `notes_sync.json`
2. При обнаружении конфликтов (удаленная версия новее) появляется диалог с выбором:
   - **Заменить локальную**: Заменить локальную версию удаленной
   - **Сохранить локально**: Оставить локальную версию без изменений

## Структура проекта

- `main.py` - Точка входа в приложение
- `gui.py` - Графический интерфейс на PyQt6
- `models.py` - Модели данных (Note)
- `database.py` - Менеджер базы данных SQLite
- `sync_manager.py` - Менеджер синхронизации
- `notes.db` - База данных SQLite (создается автоматически)
- `notes_sync.json` - Файл для синхронизации (создается автоматически)
- `notes_app.log` - Лог-файл приложения

## Расширение функционала

Приложение легко расширяется:

- **Теги**: Добавьте поле `tags` в модель `Note` и таблицу базы данных
- **Категории**: Добавьте поле `category` аналогично тегам
- **REST API синхронизация**: Укажите `sync_url` в `SyncManager` для синхронизации с сервером
- **Экспорт/Импорт**: Добавьте методы экспорта в различные форматы (Markdown, PDF и т.д.)
